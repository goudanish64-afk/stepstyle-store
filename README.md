# StepStyle Store

A modern responsive e-commerce shoe store website built using HTML, CSS, and JavaScript.

## 📌 Project Overview

StepStyle Store is a multi-page business website designed for showcasing premium footwear products. The project demonstrates responsive web design, navigation between pages, contact form validation, and modern UI concepts.

This project was created as part of a Complete Web Development Project assignment.

---

# 🚀 Features

- Responsive multi-page website
- Modern shoe store landing page
- Navigation between pages
- Product showcase section
- About page with company information
- Services/Shop page
- Contact form with validation
- Dark mode toggle
- Mobile-friendly layout
- Image optimization

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
